# ナットチェッカー ランディングページ

大型トラックの脱輪事故を防止する安全装置「ナットチェッカー」の公式ランディングページ

## 🎯 プロジェクト概要

運送業界の経営者層（50〜60代）をターゲットとした、B2B向けの信頼感と誠実さを重視したランディングページです。

### ターゲット
- 運送会社の経営者・社長
- 大型トラック車両管理責任者
- 安全管理担当者

### デザインコンセプト
- **信頼感**: 誠実で清潔感のあるデザイン
- **わかりやすさ**: 50〜60代でも読みやすい大きめの文字サイズ
- **レスポンシブ**: スマホ・タブレット・PCすべてに対応

## 🎯 最新の更新内容

### バナースライダーを追加 🎉

ページ最上部（ヘッダーの下）に、自動スライドするバナーセクションを実装しました。

**バナースライダーの特徴:**
- **自動再生**: 5秒ごとに自動で次のスライドに切り替わります
- **手動操作**: 左右の矢印ボタンでスライドを切り替え可能
- **ドットナビゲーション**: 下部のドットをクリックして特定のスライドにジャンプ
- **キーボード対応**: 矢印キー（←→）でスライド切り替え
- **スワイプ対応**: スマホでスワイプ操作が可能
- **ホバー一時停止**: マウスを乗せると自動再生が一時停止
- **完全レスポンシブ**: デスクトップ、タブレット、スマホに対応

2. banner2.png - 「目視で確実に分かるナットチェッカー」

## 🚀 使用方法

### 1. ファイルを開く

最も簡単な方法は、`index.html` をブラウザで直接開くことです：

- Windows: `index.html` をダブルクリック
- または右クリック → プログラムから開く → お好みのブラウザ

### 2. ローカルサーバーで開く（推奨）

より本番環境に近い状態でテストする場合：

#### Python を使用する場合
```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

#### Node.js を使用する場合
```bash
npx http-server -p 8080
```

その後、ブラウザで `http://localhost:8080` を開きます。

### 3. 公開する場合

以下のサービスで簡単に公開できます：

- **Netlify**: ドラッグ&ドロップで即座に公開
- **Vercel**: GitHub連携で自動デプロイ
- **GitHub Pages**: 無料で静的サイトをホスティング

## 🎨 デザイン特徴

### カラースキーム

- **プライマリーカラー**: `#DC2626` (赤) - 緊急性と重要性を表現

### レスポンシブブレークポイント

- **デスクトップ**: 1024px 以上
- **タブレット**: 768px - 1024px
- **モバイル**: 〜767px

## 🔧 カスタマイズ方法

### 色の変更

`style.css` の `:root` セクションで色を変更できます：

```css
:root {
    --primary-red: #DC2626;      /* メインカラー */
    --dark-gray: #1F2937;         /* テキストカラー */
    --bg-gray: #F9FAFB;           /* 背景カラー */
}
```

### フォントの変更

`style.css` の `--font-base` を変更：

```css
--font-base: 'Noto Sans JP', sans-serif;
```

### 画像の差し替え

`index.html` 内の `<img>` タグの `src` 属性を変更してください。
